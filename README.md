## 构建 
### Linux 
```shell
git clone --recursive https://github.com/a1ive/grub2-filemanager.git
cd grub2-filemanager
./build.sh
```

## 启动 
**不要使用 memdisk 加载 grubfm.iso !**  
### GRUB4DOS 
```
map --mem /grubfm.iso (0xff)
map --hook
chainloader (0xff)
```
### GRUB 2
```
if [ "${grub_platform}" = "pc" ]; then
  linux /loadfm  
  initrd /grubfm.iso  
else
  chainloader /grubfm.efi
fi
```
### rEFInd 
```
loader /grubfm.efi
```
### Systemd-boot 
```
efi /grubfm.efi
```
